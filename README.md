# 线性表可视化展示系统

一个基于Qt的线性表（数组和链表）可视化展示工具，支持初始化、增加、删除、修改操作，并带有流畅的动画效果。

## 功能特性

- ✅ **数组可视化**：动态展示数组结构
- ✅ **链表可视化**：动态展示链表结构
- ✅ **初始化操作**：创建指定大小的数组或链表
- ✅ **添加元素**：在指定位置添加元素
- ✅ **删除元素**：删除指定位置的元素
- ✅ **修改元素**：修改指定位置的元素值
- ✅ **动画效果**：所有操作都带有流畅的动画反馈

## 编译要求

- Qt 5.12 或更高版本（Qt 5.x系列）
- CMake 3.5 或更高版本
- C++11 或更高版本的编译器

## 编译步骤

### 方法一：使用 qmake（推荐，Qt 5.12常用方式）

```bash
qmake LinearTableVisualization.pro
make          # Linux/macOS
# 或
nmake         # Windows (MSVC)
# 或
mingw32-make  # Windows (MinGW)
```

### 方法二：使用 CMake

#### Windows (使用 Visual Studio)

```bash
mkdir build
cd build
cmake .. -G "Visual Studio 16 2019"
cmake --build . --config Release
```

#### Windows (使用 MinGW)

```bash
mkdir build
cd build
cmake .. -G "MinGW Makefiles"
cmake --build .
```

#### Linux/macOS

```bash
mkdir build
cd build
cmake ..
make
```

## 使用方法

1. **选择数据结构类型**：在顶部的下拉框中选择"数组"或"链表"

2. **初始化**：
   - 设置"大小"参数（1-20）
   - 点击"初始化"按钮

3. **添加元素**：
   - 输入元素值
   - 设置索引位置
   - 点击"添加元素"按钮

4. **删除元素**：
   - 设置要删除的索引位置
   - 点击"删除元素"按钮

5. **修改元素**：
   - 输入新值
   - 设置要修改的索引位置
   - 点击"修改元素"按钮

## 界面说明

- **控制面板**：包含类型选择、初始化、添加、删除、修改按钮
- **输入面板**：包含大小、索引、值的输入控件
- **可视化区域**：实时显示数组或链表的结构和状态

## 动画效果

- 添加元素时：元素会高亮显示并带有缩放动画
- 删除元素时：元素会高亮后消失
- 修改元素时：元素会高亮并更新值
- 所有动画持续约500毫秒，使用缓动曲线实现流畅效果

## 技术实现

- **Qt Widgets**：用于UI界面
- **QPropertyAnimation**：实现动画效果
- **自定义绘制**：使用QPainter绘制数据结构的可视化
- **CMake**：项目构建系统

## 注意事项

- 数组添加元素时，目标位置必须为空
- 链表可以在任意位置插入元素
- 索引从0开始
- 所有操作都有边界检查，超出范围会显示警告
